GIT

Git is a Version Control System (VCS).

It's like a time machine for your code. It helps you:

  • Save project history.
  • Track changes.
  • Undo mistake.
  • Work on feature safely.
  • Collaborate with other developers.

Installing

Check if is installed:

git --version

Install git

sudo apt install git

Set up

Set up your name and email:

            git congit --global user.name "Your name"
            git congit --global user.name email "your.email@example.com"

Check your configuration:

git config --list

Now you are ready to use git

Creating a repository

A repository is a folder that contains your project and all the git history.

Create a folder

mkdir my-project

Navigate to your folder

cd my-project

initialize Git

git init

Check status:

git statue

Add file to staging area

git add dummy.txt

Add every change

git add .
