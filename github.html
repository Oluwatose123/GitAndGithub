GITHUB

Github is awebsite where Git projects are stored online.

Git is a tool. Github is the cloud storage for git projects.

Github allows you:

  • Backup your code.
  • Share project.
  • Build project.
  • Collaborate.

Push Project To Github

git push -u origin main

Push project To Github

Step 1: Create a Repository

  1. Go to Github.
  2. Click New Repository.
  3. Enter a Repository (e.g my-first-project).
  4. Click Create Repository.
  5. Leave the page open.

step 2:Open your project in VS Code.

Open your project that contain your project files.

Step 3:Open Terminal in VS Code (ctrl + j).

step 4:initialize Git

git init

This tells git to start tracking this project.

Step 5: Check Files

git status

You will see untracked files.

Step 6: Stage files

git add .

add: means move files to the staging area.

.:means everything.

The staging area is like a waiting room.

Step 7:Commit Files

git commit -m "first commit"

Commit: means save a snapshot.

-m: means message.

Example: git commit -m "just create a footer"

Step 8:Connect Local Project to Github

Copy the URL from Github. Example https://github.com/yourname/my-first-project.git

Run this in your terminal:

git origin https://github.com/yourname/my-first-project.git

remote: means online

Step 9: Rename Branch to Main

git branch -M main

Branch: means line of development

Main: means primary branch.

Step 10: Push to Github

git push -u origin main

Push: upload

Origin: github repository

main: branch main

Go and refresh your github, these files will now appear on Github.

What do i do every time i make changes?

Supose you edit your files. Or each time you edit your file.

  1. change changes: git status
  2. Stage changes: git add .
  3. Commit changes: git commit -m "Updated hompage"
  4. Push change: git push

That's it. Repeat these four cycle whenever you update a project.
